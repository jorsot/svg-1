Assignment 02 - SVG Collage - J.J. Lynn

This is part of the fictional city of "Las Zanahorias," which contains buildings and places that my films have taken place in. Below is a radio station integral to the plot of "Midnight Macabre." The image below was created in Adobe Illustrator and exported to an .svg format. The car animations are animated using the animation tags in the SVG file while the day/night cycle and the headlights are styled using CSS animations.

Unfortunately, I was unable to get blend modes to work properly, so the night sky does not blend well with the components underneath, leaving a blue haze instead of the impression of night. I also had an issue with the rectangle I was using for the night effect covering up the link. To solve this, I created another path that perfectly outlined the building and placed it on top of everything else. By making this shape transparent, it retains the link shape of the building while also remaining on top of the layer stack.

Clicking on the station will link the user to the video on YouTube.
