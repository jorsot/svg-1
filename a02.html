Assignment 02 - SVG Collage - J.J. Lynn

This is part of the fictional city of "Las Zanahorias," which contains buildings and places that my films have taken place in. Below is a radio station integral to the plot of "Midnight Macabre." The image below was created in Adobe Illustrator and exported to an .svg format. The car animations are animated using the animation tags in the SVG file while the day/night cycle and the headlights are styled using CSS animations.

FORK: The animation has been changed to reflect the massive panic in the city of Las Zanahorias, everyone is rushing to get their affairs in order, hence the fast drivers.

Clicking on the station will now link the user to a new, appropriate video to help with the citizen's troubles on YouTube.
